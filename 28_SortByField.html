Sort by field

Дан массив объектов 

          
let users = [
  { name: "John", age: 20, surname: "Johnson" },
  { name: "Pete", age: 18, surname: "Peterson" },
  { name: "Ann", age: 19, surname: "Hathaway" }
];

Для сортировки элементов в нем можно использовать метод массива sort 

          
// by name (Ann, John, Pete)
users.sort((a, b) => a.name > b.name ? 1 : -1);

// by age (Pete, Ann, John)
users.sort((a, b) => a.age > b.age ? 1 : -1);

1. Напишите функцию byField, которая поможет сделать этот код более читаемым.

        
users.sort(byField('name'));
users.sort(byField('age'));

2. Результат выведите в виде HTML таблицы с помощью функции renderTable.
